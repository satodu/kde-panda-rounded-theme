# Panda Rounded Light

Versão clara do **Panda Rounded Theme** — mesmos cantos arredondados, fundo claro.

## Instalação

O tema já está em:

```bash
~/.local/share/plasma/desktoptheme/Panda-theme-light
```

1. Abra **Configurações do Sistema** → **Aparência** → **Estilo Plasma**
2. Selecione **Panda Rounded Light** e aplique
3. Recomendado: use também um *esquema de cores* claro (ex.: Breeze Light)
