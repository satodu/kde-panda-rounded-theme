# Klassy (decoração de janelas)

O **Klassy** é um plugin externo (`klassy-bin` no AUR / [paulmcauley/klassy](https://github.com/paulmcauley/klassy)).
Não está embutido neste repositório — instale-o antes de aplicar o tema.

## O que este repo guarda

| Ficheiro | Destino | Conteúdo |
|----------|---------|----------|
| `klassyrc` | `~/.config/klassy/klassyrc` | Tua configuração (margens, botões, bordas Dolphin, etc.) |

Presets oficiais (`windecopresetsrc`) e ícones `klassy` / `klassy-dark` vêm com o pacote — não versionamos.

## Instalação

O `../install.sh` copia o `klassyrc` e aponta o Look-and-Feel para `org.kde.klassy`.
